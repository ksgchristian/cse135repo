**Tool Used:** Simple Analytics  

**Why this tool:**  
Simple Analytics was chosen for its strong focus on privacy. It does not use cookies, does not track personal data, and is GDPR-compliant by default. **Analysis:**  
Compared to more invasive analytics platforms, Simple Analytics prioritizes ethical data collection. It is easy to set up, lightweight, and aligns well with privacy-conscious web development practices while still giving meaningful usage metrics.
